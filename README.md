# Moonsoup Fuse-ext2

## How do I install these formulae?

`brew install moonsoup/fuse-ext2/<formula>`

Or `brew tap moonsoup/fuse-ext2` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "moonsoup/fuse-ext2"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
